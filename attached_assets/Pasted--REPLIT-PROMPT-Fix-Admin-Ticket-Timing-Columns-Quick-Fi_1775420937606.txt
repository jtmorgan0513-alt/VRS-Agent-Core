### 2. Remove the Handle Time cell and fix Queue Wait logic

Find the three timing `<TableCell>` elements (around line 1072-1090). Replace ALL THREE with just these two:

```tsx
{/* Queue Wait: how long the ticket sat before an agent claimed it */}
<TableCell className="text-sm font-medium text-right whitespace-nowrap">
  {ticket.ticketStatus === "queued"
    ? <span className="text-amber-600 dark:text-amber-400">{getTimeInStatus(ticket.createdAt)} ⏳</span>
    : ticket.claimedAt
    ? getTimeInStatus(ticket.createdAt, ticket.claimedAt)
    : "—"
  }
</TableCell>

{/* Total Time: end-to-end from submission to resolution (or live if still open) */}
<TableCell className="text-sm text-right whitespace-nowrap">
  {["completed", "approved", "rejected", "rejected_closed", "invalid"].includes(ticket.ticketStatus)
    ? getTimeInStatus(ticket.createdAt, ticket.statusChangedAt)
    : <span className="text-muted-foreground">{getTimeInStatus(ticket.createdAt)} ⏳</span>
  }
</TableCell>
```

**Logic explained:**
- **Queue Wait**: If ticket is still queued → show live timer with ⏳. If claimed → show `createdAt` to `claimedAt`. If old ticket without `claimedAt` data → show "—".
- **Total Time**: If resolved → show `createdAt` to `statusChangedAt` (final number). If still open → show live timer with ⏳.
