Add daily RGC code system for Sears Protect authorizations. Include:
1. RGC Code Format:

Format: RGC followed by 5 digits (e.g., RGC12345)
Regex validation: ^RGC\d{5}$
Display with uppercase RGC prefix always

2. Database:

Use existing daily_rgc_codes table:
sqlid, code, valid_date, created_by, created_at

Add to users table: last_rgc_code_entry DATE — Track when agent last entered the code

3. Admin: Set Daily RGC Code

Add "Daily RGC Code" section to Admin dashboard
Form with:

Code input field with placeholder "RGC12345"
Auto-prefix "RGC" — agent only types the 5 digits
Date picker (defaults to today)
"Set Code" button


Validation: Must be exactly 5 digits after RGC prefix
Shows current active code and who set it
Can set tomorrow's code in advance
API: POST /api/admin/rgc-code — Creates/updates code for a date
API: GET /api/admin/rgc-code?date={date} — Get code for specific date

4. VRS Agent: Daily Code Entry Prompt

On login, check if last_rgc_code_entry < today
If true, show modal before accessing dashboard:

Title: "Enter Today's RGC Code"
Input field with "RGC" prefix shown, agent types 5 digits
Placeholder: "12345"
"Submit" button


On submit:

Validate format (5 digits)
Validate code matches today's daily_rgc_codes entry
If valid: Update last_rgc_code_entry = today, dismiss modal, proceed to dashboard
If invalid: Show error "Code does not match. Please check with your admin."
Store validated code in session/state for the day


Agent cannot access dashboard until code is entered correctly

5. Stage 2 Authorization: Auto-Populate RGC

In the Stage 2 action panel, for Sears Protect submissions:

Show "RGC Code" field (read-only, pre-filled with today's code, e.g., RGC12345)
Show "Authorization Code" field (agent enters this from warranty provider)


Both codes sent to technician in the SMS:
VRS Authorization for SO#{service_order}
RGC Code: RGC12345
Auth Code: {auth_code}
Enter both codes in TechHub to complete the job.


6. Update Submissions Table:

rgc_code VARCHAR(50) — Store the RGC code used
Auto-populate from session when Stage 2 is completed

7. VRS Agent Header/Sidebar:

Display "Today's Code: RGC12345" prominently
"Update Code" link if they need to re-enter (edge case: code changed mid-day)

8. Edge Cases:

If no RGC code set for today: Show VRS agents message "Daily code not set. Contact admin." Block Stage 2 processing.
If agent's session expires mid-day: Re-prompt for code on next action
If admin changes code mid-day: Notify agents to re-enter code

9. Validation Messages:

Invalid format: "RGC code must be RGC followed by 5 digits (e.g., RGC12345)"
Code mismatch: "Code does not match today's RGC. Please verify with your admin."
No code set: "No RGC code has been set for today. Please contact an administrator."
